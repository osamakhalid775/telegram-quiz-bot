# بوت أسئلة وألغاز جماعي لتليجرام

بوت تفاعلي يدعم الألعاب الجماعية في المجموعات بوضع "أول من يجب". يمكنك بدء لعبة أسئلة اختيار من متعدد أو ألغاز، ويتنافس الأعضاء على الإجابة.

## الميزات
- أسئلة متعددة المجالات (جغرافيا، علوم، أدب، ألغاز)
- نظام نقاط وتنافس جماعي
- ترتيب اللاعبين في المجموعة
- إدارة الألعاب من قبل المشرفين

## كيفية النشر على Render

1. انسخ هذا المشروع إلى مستودع GitHub.
2. سجل دخولك إلى [Render](https://render.com).
3. أنشئ خدمة جديدة من نوع **Web Service** (أو **Worker** إذا أردت).
4. اربط المستودع الخاص بك.
5. في إعدادات الخدمة:
   - **Name**: اختر اسماً (مثل telegram-quiz-bot)
   - **Environment**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python main.py`
6. أضف متغير البيئة `BOT_TOKEN` وقيمته التوكن الخاص بك من BotFather.
7. اضغط **Create Web Service**.

الآن البوت يعمل! أضفه إلى مجموعتك وابدأ اللعب.

## الأوامر
- `/play` - بدء لعبة جديدة
- `/endgame` - إنهاء اللعبة (للمشرفين)
- `/score` - عرض نقاطك
- `/leaderboard` - عرض ترتيب المجموعة
- `/help` - المساعدة

## ملاحظات
- البوت يحتاج لصلاحية قراءة الرسائل في المجموعة.
- الأسئلة الحالية افتراضية ويمكنك تعديلها في ملف `main.py` (قائمة `QUESTIONS`).
- للتوسع، يمكن استبدال التخزين في الذاكرة بقاعدة بيانات مثل SQLite أو PostgreSQL.
