# بوت أسئلة وألغاز جماعي لتليجرام (مع SQLite)

بوت تفاعلي يدعم الألعاب الجماعية في المجموعات. يتم تخزين النقاط بشكل دائم في قاعدة بيانات SQLite، والأسئلة محملة من ملف JSON.

## الميزات
- أسئلة متعددة المجالات من ملف JSON.
- نظام نقاط دائم لكل مستخدم (إجمالي وفي كل مجموعة).
- ترتيب اللاعبين في المجموعة.
- إدارة الألعاب من قبل المشرفين.

## كيفية النشر على Render

1. انسخ هذا المشروع إلى مستودع GitHub.
2. في Render، أنشئ خدمة جديدة (Web Service) واربط المستودع.
3. استخدم الإعدادات:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python main.py`
4. أضف متغير البيئة `BOT_TOKEN`.
5. تأكد من رفع ملف `questions.json` إلى نفس المسار.

البوت سينشئ ملف `quiz_bot.db` تلقائياً عند أول تشغيل.

## الأوامر
- `/play` - بدء لعبة جديدة
- `/endgame` - إنهاء اللعبة (للمشرفين)
- `/score` - عرض نقاطك الإجمالية
- `/score_group` - عرض نقاطك في هذه المجموعة
- `/leaderboard` - عرض ترتيب المجموعة
- `/help` - المساعدة
